Why PHP, not plain HTML

Stop pasting the same navbar into every file.

Static HTML makes you copy the chrome onto every page. PHP keeps the same simple, build-free workflow and lets you write it once, then loop over your data. Same easy hosting. Far less to maintain.

<!-- the boring way -->
<nav> ... copy ... </nav>
<nav> ... paste ... </nav>

// the elegance way
<?php require('includes/header.php') ?>
